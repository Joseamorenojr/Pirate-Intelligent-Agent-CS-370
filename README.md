In this project, I developed a pirate intelligent agent using reinforcement learning concepts, specifically Q-learning combined with neural networks. The goal of the project was to train an agent to navigate an environment efficiently while maximizing rewards and minimizing penalties. I was initially provided with a structured Jupyter Notebook that included the environment setup, predefined helper functions, and some foundational reinforcement learning components such as the state space configuration, reward system, and episode loop framework. The starter code also included scaffolding for training the agent and evaluating performance.

The core logic of the intelligent agent, however, was developed by me. I implemented the neural network model that approximated the Q-function, defined the architecture layers, selected activation functions, and configured hyperparameters such as learning rate, discount factor, and exploration rate. I also implemented the epsilon-greedy exploration strategy, the experience replay mechanism, and the training loop that updated Q-values based on observed transitions. Additionally, I tuned parameters and tested performance to ensure the agent learned an optimal or near-optimal policy. Through debugging and iterative testing, I improved convergence stability and overall agent performance.
